void gpio_port_set ( uint32_t  port,
uint32_t  mask 
)

set port pins state by mask

Parameters
portGPIO port number (0 .. GPIO_PORTS_CNT)
maskGPIO pins mask to set (0 .. 0xFFFFFFFF)

mask examples:

mask = 0xFFFFFFFF (0b11111111111111111111111111111111) means set all pins state to 1 (HIGH)
mask = 0x00000001 (0b1) means set pin 0 state to 1 (HIGH)
mask = 0x0000000F (0b1111) means set pins 0,1,2,3 states to 1 (HIGH)
Return values
none
Examples:
mod_gpio.c.

uint32_t  mask 
)

clear port pins state by mask

Parameters
portGPIO port number (0 .. GPIO_PORTS_CNT)
maskGPIO pins mask to clear (0 .. 0xFFFFFFFF)

mask examples:

mask = 0xFFFFFFFF (0b11111111111111111111111111111111) means set all pins state to 0 (LOW)
mask = 0x00000003 (0b11) means set pins 0,1 states to 0 (LOW)
mask = 0x00000008 (0b1000) means set pin 3 state to 0 (LOW)
